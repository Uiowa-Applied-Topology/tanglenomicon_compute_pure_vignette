# Load Job Queue

## Primary actor

- Start up

## Trigger

Faktory starts with the job queue empty.

## Goal

The job queue is filled.

## Preconditions

The tangle long term storage are connected.

## Scenario

1. Faktory starts
2. Faktory has an empty queue.
3. Tangles are paginated into jobs.
4. New jobs are reported to Faktory.
