# Compute on Pages

## Primary actor

N/A

## Trigger

An upstream actor wants to compute over a page of tangles.

## Goal

Run a core computation on a page of tangles.

## Preconditions

The long term storage has been initialized.

## Scenario

1. An actor initiates a use case that retrieves a page of tangles
2. The core computation is executed on each tangle in the page.
